W55MH32 Reference Codes

The W55MH32 provides the networking, peripheral and security encryption Example.

The W55MH32 Standard Peripherals library provides a rich set of examples covering the main features of each peripheral. All the examples are based on W55MH32L-EVB and W55MH32Q-EVB. Only source files are provided for each example and the user can tailor the provided project template to run the selected example with the user's preferred tool chain.

  • Peripherals​
