1x2 FACT RST (J4)​

Pin NumberSignalDescription
1FACT RSTFactory Reset Signal input
Drive this pin to Low for at least 5.0s then WIZ510SR-RP will be reset to factory settings
2GNDSystem Ground
  • J4 is not mounted

Switch Specification​

Reset Switch (SW1)​

  • When Push SW1, WIZ510SR-RP will be reset

BOOTSEL Switch (SW2)​

  • Push SW2 and reset(or power on),then RP2040 will enter USB Mass Storage Device mode

    Then RP2040 can be re-programming

HW TRIG Switch (SW3)​

  • Slide SW3 Left : AT command
  • Slide SW3 Right : Gateway Mode
  • Changes will be reflected after reset
