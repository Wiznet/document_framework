WIZ550web

Overview​

WIZ550web is an embedded Web server module based on WIZnet W5500 hardwired TCP/IP chip, Users can control & monitor the 16-configurable digital I/O and 4-ADC inputs on module via web pages. WIZnet provides firmware & web page examples that base on WIZ550web for user to customize their own objective.
