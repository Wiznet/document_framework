WIZPoDL-P1

WIZPoDL-P1


By using WIZPoDL-P1, you can efficiently power external applications within the 7W range without the need for a power cable, not only for Ethernet but also for various external applications.

Features​

  • Support PoDL Class 10, 11, 13
  • Support PoDL Detaction
  • Support PoDL SCCP protocol
  • Output Isolation
  • Internal build in 2 channel bridge rectifiers
  • auto polarity correction
