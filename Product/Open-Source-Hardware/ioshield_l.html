ioShield-L

Overview‚Äã

ioShield-L is an Ethernet BoosterPack for MSP430 LaunchPad development board(üåéMSP-EXP430G2, TI). The ioShield-L allows a MSP430 LaunchPad board to connect to the internet. It consists of WIZ550io for ethernet connectivity and an interface board to LaunchPad.

MSP430 LaunchPad‚Äã

The MSP-EXP430G2 low-cost experimenter board called LaunchPad is a complete development solution for the Texas Instruments MSP430G2xx Value Line series. The integrated USB-based emulator offers all the hardware and software necessary to develop applications for all MSP430G2xx series devices.

For more details, please refer to üåéMSP430 LaunchPad page in üåéTI official website.

Development Environment‚Äã

The MSP-EXP430G2 can be used with IAR Embedded Workbench‚Ñ¢ Integrated Development Environment (IDE), Code Composer Studio‚Ñ¢(CCS) IDE or Energia (open-source electronics prototyping platform based on the Wiring and Arduino framework) to write, download, and debug applications. Now we provides libraries for IAR and Energia, except CCS. Libraries for CCS will be released soon.

ioShield-L supported devices‚Äã

  • Board : MSP-EXP430G2 Rev. 1.5 and later. (for H/W UART support)
  • MCU : MSP430G25x3IN20 (PDIP20 package, 16kB Flash, 512B RAM) and later.
