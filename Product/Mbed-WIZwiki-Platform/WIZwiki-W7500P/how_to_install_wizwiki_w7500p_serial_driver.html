How to install WIZwiki-W7500p serial driver

Overview#

CMSIS-DAP of WIZwiki-W7500p has two functions. The first function is firmware writing to the W7500p (Target MCU) through USB Interface. The second function is to receive and transmit data in the UART1 of W7500p using the USB Interface.
The first function is used without the driver, but the second function is used after install the driver.
CMSIS-DAP Interface of WIZwiKI-W7500 has shows as the photo below.

Serial driver installation#

The driver could be found on the LINK below, click on the "Download latest driver" can download the latest driver.
