## Features

<ul>
  <li>Supports <b>IPv4/IPv6 Dual Stack</b></li>
  <li>Hardware-based Internet protocols: TCP, UDP, ICMPv6, ICMPv4, IGMP, MLDv1, ARP, PPPoE</li>
  <li>8 independent sockets with 32 KB internal TX/RX buffer</li>
  <li>Supports SOCKET-less commands: ARP, PING, ICMPv6 (DAD, NA, RS)</li>
  <li>Auto-negotiation and Auto-MDIX for 10/100 Mbps operation</li>
  <li>Ethernet Power-down & Main Clock Switching for low power</li>
  <li>Supports Wake-on-LAN (UDP)</li>
  <li>High-speed SPI interface (Mode 0/3)</li>
  <li>3.3 V operation with 5 V I/O tolerance</li>
  <li>2 × 6-pin headers (2.54 mm pitch)</li>
  <li>Operating temperature: −40 °C to +85 °C</li>
</ul>

### Driver
The ioLibrary means "Internet Offload Library" for WIZnet chip. It includes drivers and application protocols. The ioLibrary can be used for W5100S and other chips's application designs. These will be updated continuously.

| **Resource** | **Description** |
|--------------|-----------------|
| [![link](\img\link.png) ioLibrary_Driver](https://github.com/Wiznet/ioLibrary_Driver) | Official WIZnet driver library for W6100 and other chips |


      :::info

The ioLibrary_Driver is an MCU-independent library for WIZnet W5x00, W6x00 chips .
It provides implementations of essential TCP/IP services, enabling developers to build network applications with minimal MCU dependencies.
      ### Supported services
DHCP, DNS, MQTT, SNTP, TFTP, HTTP Server

      :::
