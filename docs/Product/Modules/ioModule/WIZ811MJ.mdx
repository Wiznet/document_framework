## Features
![](/img/products/wiz811mj/wiz811MJ.jpg)
<ul>
  <li>Plug-in network module integrating <b>W5100</b> chip, transformer, and RJ45 MAG-JACK</li>
  <li>No external design required for transformer or PHY</li>
  <li>Fast evaluation for <b>W5100</b> and MCU-based systems</li>
  <li>Supports <b>MCU BUS</b> and <b>high-speed SPI interface</b></li>
  <li>Supports 4 independent hardware sockets</li>
  <li>Operating temperature: −40 °C ~ +85 °C</li>
  <li>Compact plug-in module for rapid development</li>
</ul>
