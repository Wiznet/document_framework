    <p>
      The <b>WIZ850io</b> is a compact network module that integrates the <b>W5500</b> hardwired TCP/IP chip with a built-in PHY, transformer, and RJ45 MAG-JACK.  
      It is fully compatible with <b>WIZ820io</b> hardware, providing developers with a seamless upgrade path and faster network performance.  
      Ideal for rapid prototyping and Internet-enabled embedded systems, it requires no additional interface design between the W5500 and transformer.

## Features

<ul>
  <li>Compact plug-in Ethernet module integrating <b>W5500</b> with transformer and RJ45 MAG-JACK</li>
  <li>Hardware compatible with <b>WIZ820io</b></li>
  <li>No hardware design required for PHY or transformer</li>
  <li>Supports <b>high-speed SPI interface (Mode 0/3)</b></li>
  <li>Power-down and Wake-on-LAN support</li>
  <li>Small form factor: <b>20 × 25 mm</b></li>
  <li>1×6-pin headers (2.54 mm pitch)</li>
  <li>Operating temperature: −40 °C ~ +85 °C</li>
</ul>
