    <p>
      The <b>WIZ610MJ</b> is a compact Ethernet module that integrates the <b>W6100</b> hardwired TCP/IP chip with an embedded PHY and an RJ45 MAG-JACK. 
      It supports both <b>IPv4 and IPv6 Dual Stack</b>, enabling flexible networking for modern embedded systems. 
      By combining all essential Ethernet hardware in one module, it eliminates complex PCB design for the transformer and RJ45 interface.
      The WIZ610MJ is ideal for developers who want to implement network-enabled systems quickly and reliably.

## Features

<ul>
  <li>Supports <b>IPv4/IPv6 Dual Stack</b></li>
  <li>Hardwired Internet protocols: TCP, UDP, ICMPv6, ICMPv4, IGMP, MLDv1, ARP, PPPoE</li>
  <li>8 independent sockets with 32 KB internal TX/RX buffer</li>
  <li>SOCKET-less commands for ARP, PING, and ICMPv6 (DAD, NA, RS)</li>
  <li>High-speed SPI interface (Mode 0/3)</li>
  <li>Parallel bus mode with 8-bit data and 2 address lines</li>
  <li>Auto-negotiation and Auto-MDIX (for 10/100 Mbps)</li>
  <li>Ethernet Power-down & Main Clock Switching for low-power operation</li>
  <li>Supports Wake-on-LAN (UDP)</li>
  <li>3.3 V operation with 5 V I/O tolerance</li>
  <li>2 × 10-pin headers (2.54 mm pitch)</li>
  <li>Operating temperature: −40 °C to +85 °C</li>
</ul>
