      It supports <b>QSPI and high-speed operation up to 150 MHz</b>, enabling data throughput over 80 Mbps.

## Features

<ul>
  <li>Supports <b>Hardwired TCP/IP protocols</b>: TCP, UDP, IPv6, IPv4, ICMPv6, ICMPv4, IGMP, MLDv1, ARP, PPPoE</li>
  <li>Dual Stack <b>IPv4/IPv6</b> support</li>
  <li>High-speed operation with <b>150 MHz system clock</b></li>
  <li>Up to <b>80 Mbps throughput</b> via QSPI (4-bit data interface)</li>
  <li>8 independent sockets with total <b>64 KB SRAM</b> (4 KB TX/RX per socket)</li>
  <li>Supports SOCKET-less commands (ARP, ICMPv6 for DAD, NA, RS)</li>
  <li>Integrated transformer and RJ45 MAG-JACK</li>
  <li>Supports Ethernet PHY power down & clock switching for low power</li>
  <li>Supports <b>Wake-on-LAN (UDP)</b></li>
  <li>Supports <b>SPI and QSPI</b> interfaces</li>
  <li>10BaseT / 10Base-Te / 100BaseTX Ethernet PHY integrated</li>
  <li>Auto-Negotiation and Auto-MDIX supported</li>
  <li>3.0 V operation (5 V I/O tolerant)</li>
  <li>Network LEDs for link, speed, duplex, and activity</li>
  <li>Operating temperature: −40 °C to +85 °C</li>
</ul>
