WIZMC-T1L can serve as a bridge between existing communication environments and newly established SPE (Single Pair Ethernet) networks.
Building an SPE communication environment can greatly enhance the efficiency of your project — improving power line utilization, increasing communication speed, and reducing costs. We are confident that SPE is a far superior protocol compared to conventional infrastructures.

However, even if you decide to migrate your entire communication environment to SPE, the biggest challenge will be how to connect with your existing protocol-based equipment.
WIZMC-T1L eliminates all those concerns by providing seamless connectivity between the two environments.

-----

## Features

- Module status monitoring and configuration available through 2nd Ethernet based on W55RP20
- Built-in external 10Base-T1L PHY chip, capable of communication **up to 1.2 km over 2 wires (1 pair)**
- Status pins: Power status / PHY link status / TCP connection status
- Operating temperature: **-40 to 80°C**, industrial level
- 2kV ESD protection circuit
- **Input voltage : 5–40V**
- **Power consumption : 0.5–1W**
