# WIZ750SR-T1L

<div className="main-card">
  <img src="/img/products/wiz750sr-t1l/wiz750sr-t1l_3d" alt="WIZ750SR-T1L" />
  <div>
    <h3>WIZ750SR-T1L</h3>
    <br />
    <p>
      WIZ750SR-T1L is a compact sized serial to Single Pair Ethernet module using the W7500 Ethernet MCU (SoC based on ARM Cortex-M0 architecture) with External 10Base-T1L PHY(Motorcomm).
    </p>
  </div>
</div>

WIZ750SR-T1L is a protocol converter that transmits data sent by serial equipment as TCP/IP data, and converts the TCP/IP data received through the network back into serial data to the serial equipment. It complies with industrial temperature standard.

In addition, WIZ750SR-T1L contains an 4-pin GPIO expansion port that can be controlled freely by the user. (analog input or digital in/out)

**WIZ750SR-T1L is the best fit solution to control user's serial device via 10Base-T1L Ethernet network.**

**Also compatible with WIZ750SR** is designed to be used. Users can use various kinds of boards if only changing board selection in
firmware. (ex WIZ750SR, WIZ750SR-100, WIZ750SR-105, WIZ750SR-110, WIZ750SR-T1L)

-----

## Features

- Serial to 10Base-T1L Ethernet Module based on **W7500** MCU (TTL level)
- Included External 10Base-T1L PHY chip. Capable of communication up to 1.2 km over 2 wires (1 pair)
- **10Mbps T1L Ethernet & Up to 230kbps serial speed**
- Status pins: PHY link status / TCP connection status
- Serial signals(TTL)
    - WIZ750SR-T1L module: TXD, RXD, RTS, CTS
    - Optional DTR/DSR signals supported (Status pins or DTR/DSR pins
      selectable)
- Separate **Data UART** and **Debug UART** port
- Supports a search password 
- Configuration is possible via serial AT commands or via
  Configuration tool 
- Built in Supervisor IC for improvements of Reset function stability
