- IEEE802.3af compliant
- Mode A(Endspan), Mode B(Midspan)
- Wide input voltage range 40Vdc ~ 60Vdc
- High DC/DC convertion efficiency
- Non-Isolation
- Internal build in 2 channel bridge rectifiers
